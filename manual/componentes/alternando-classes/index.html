Alternando classes

É possível alternar entre 2 classes no seu projeto utilizando LocawebStyle.

As vezes surge a necessidade em alternar classes ao clicar em um determinado elemento.

É muito comum esse comportamento ser utilizado em alternância de ícones, agora ficou muito fácil executar essa funcionalidade.

Basta clicar abaixo e ver a interação visual

<a href="#" class="ico-heart-empty" data-classtoggle="ico-heart-empty,ico-heart-1"></a>
<a href="#" class="ico-plus-circle-1" data-classtoggle="ico-plus-circle-1,ico-minus-circle"></a>
<a href="#" class="ico-lock-1" data-classtoggle="ico-lock-1,ico-unlock-1"></a>
<a href="#" class="ico-folder" data-classtoggle="ico-folder,ico-folder-open"></a>


Atributo Descrição
class A Classe que será exibida ao carregar a página
data-classtoggle Inserimos 2 classes separadas por (,).
O primeiro é o mesma classe carregada no elemento, e o Segundo é a classe que será inserida ao ser clicado.
