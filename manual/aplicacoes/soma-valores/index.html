Soma Valores

Exibe um conjunto de radio-buttons ou checkboxes para seleção e exibe um sumário dos selecionados

O componente usa alguns atributos data no html para exibir um resumo das opçoes selecionadas.

Atributos & Classes

A lista deve estar dentro de um container com a classe ls-sum-values-container

Cada item da lista deve possuir a classe sumItem

O input radio ou input checkbox dentro do itemSum deverá possuir os atributos:

  • data-sumvalue="999": valor a ser somado e exibido.
  • data-sumlabel="TEXTO": texto a ser exibido no resumo.

O resumo, texto e soma numérica, será exibido dentro de um elemnto que possua a classe sumResume, com elementos filhos com as classes:

  • sumResumeText: exibe os textos, concatenados, das opçoes selecionadas.
  • sumResumeTotal: exibe a soma numérica, exibida de acordo com o atributo data-sumpattern="R$ {{val}}/mês"
